# Community examples worth mining

These are not direct dependencies. Use them as references for task ideas, camera layouts, dataset names, and workshop inspiration.

## Simple / relevant now

- CreatorKanata/gum-pickup-so-arm101: complete SO-ARM101 gum/candy pickup workflow with LeRobot.
- sachinkum0009/so101-imitation-learning: SO101 toothbrush pickup README with calibrate, record, train, and evaluate command examples.
- lerobot/svla_so101_pickplace: public LeRobot SO101 pick-place dataset; useful to show students what a robot dataset looks like.
- aswinkumar99/LeRobot-SO101-Pick-Place: sponge pick-and-place dataset with fixed/random object layouts.
- xinjiehu76/so101-pick-place-dataset: red cube to gray bowl dataset.

## Advanced / save for later

- legalaspro/so101-ros-physical-ai: ROS2 + policy server/inference stack.
- ros-physical-ai/demos: ROS2 + LeRobot bridge for real hardware/simulation.
- pham-tuan-binh/leslider: 7th-axis slider extension for SO-101.
- VLA/GR00T/SmolVLA demos: interesting for advanced sessions, not the first 3-hour workshop.
