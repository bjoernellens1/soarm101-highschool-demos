# Teacher checklist

## Before students arrive

- Arm base clamped and marked with tape.
- Workspace mat taped down.
- Foam/sponge objects ready.
- Bins taped down.
- Laptop charged and environment activated.
- `scripts/03_teleop_wow.sh` tested.
- Camera checked with `scripts/06_color_sort_cv.py`.
- One backup demo video ready in case hardware fails.

## Safety briefing, 60 seconds

1. Only one person drives the robot at a time.
2. Hands stay outside the follower workspace when the robot moves.
3. Objects are light and soft.
4. Teacher can unplug power at any time.
5. We debug slowly; speed is not the goal.

## Good student questions

- What changed when the object moved?
- Why did the robot miss the cube?
- What would a camera help with?
- What data would we need for the robot to learn this?
